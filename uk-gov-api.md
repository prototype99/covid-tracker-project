api that covers the [United Kingdom](uk.md). Needs input to be in [json](json.md).
##### available data:
- [cases](cases.md)
- [deaths](deaths.md)
- [upload time](upload.md)
##### ouputs with [gzip compression](gzip.md) as:
- [csv](csv.md)
- [json](json.md)
- [xml](xml.md)
##### can filter by:
- [area](area.md)
- [date](date.md)
##### supported languages are:
- [C#](c-sharp.md)
- [elixir](elixir.md)
- [javascript](js.md)
- [python](python.md)
- [R](r.md)
